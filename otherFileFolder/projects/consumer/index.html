Time series analysis of #complaints is very crucial. Here we observe how the number of complaints generally peak in mid year.

Graph 4

Number of complaints reported via different mediums


Number of complaints reported via web are the highest. It is really interesting to note that as the #complaints increase via web, the other mediums shrink.

When #complaints are soreted based on the products, we notice, mortage related complaints are the higest. This is followed by Debt collection which has the higest number of dis-satisfied consumers as they report untimely response.

Summary

California reports the highest complaints

States where most of cities are located (centres of commerce and business), report the higest #complaints. Makes sense as the top complaints are about mortgage.

Web is the preferred way to report complaints

This not only shows that web has high penetration but also has shown capabilities to resolve issues faster.

Morgate related complaints top the chart

From the analysis, it is seen that mortgages and loans are the biggest areas facing complaints.

Forecasts

Forecasts predict, the number of complainst will decrease in 2017. More happy customers.
